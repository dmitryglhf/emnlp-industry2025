# Summary

## TL;DR  
TelAgentBench is a Korean, privacy‑aware benchmark that evaluates five core agentic capabilities—Reasoning, Planning, Action (tool‑use), Retrieval‑Augmented Generation (RAG), and Instruction Following (IF)—for LLMs in telecommunications. With 1,700 synthetic instances and 23 telecom‑specific APIs, it reveals a large performance gap between models that explicitly reason (thinking‑enabled) and those that do not, offering concrete guidance for deploying LLM agents in real‑world telco services.

## Problem & Motivation  
Telecommunications operators are expanding beyond traditional MNO roles into lifestyle‑integrated services (e.g., itinerary planning, roaming plan management). Existing generic benchmarks fail to assess realistic, non‑English performance and domain‑specific constraints, leaving a gap for evaluating LLM agents in telco contexts. TelAgentBench addresses this by providing a Korean, privacy‑preserving dataset that mirrors commercial telco workflows and regulatory constraints.

## Approach  
- **Five Core Capabilities**: Reasoning, Planning, Action, RAG, IF.  
- **Synthetic Construction**: Seed instances created by linguists, expanded via paraphrasing, generative models, and cross‑review to avoid ambiguity.  
- **Domain‑Specific Environment**: 23 telecom APIs (billing, roaming, data, rate plans, family info, miscellaneous) with realistic input/output specs and a simulated customer database.  
- **Knowledge Retrieval**: Business documents (terms, manuals, FAQs) segmented and indexed; each RAG sample includes 5 documents (1 ground‑truth + 4 distractors).  
- **Instruction Following**: Korean‑specific linguistic constraints (syllable count, honorifics) and telco‑specific instructions (table formatting, sensitive info handling).  
- **Evaluation Pipeline**: Offline scoring with accuracy, pass‑rate, faithfulness metrics; no online deployment context reported.

## Experimental Setup  
| Dataset | Size | Notes |
|---------|------|-------|
| TelAgent Reason | 225 | Multi‑hop QA over telecom policy docs |
| TelAgent Plan | 200 | Adapted TravelPlanner with telco constraints |
| TelAgent Action | 757 | 23 APIs, 10 sub‑tasks, distractor functions |
| TelAgent RAG | 258 | 5 documents per sample, distractors |
| TelAgent IF | 300 | Multi‑turn Korean instructions |

**Models**: 15 leading LLMs (proprietary: Sonnet 4.5, Opus 4.1, o3, Gemini 2.5 Flash, Anthropic Sonnet 4.5, Opus 4.1, Haiku 3.5, OpenAI GPT‑5; open‑source: Llama 3.3‑70B‑Instruct).  
**Metrics**: Accuracy (Reason, Action, IF), Pass‑rate (Plan), Faithfulness (RAG).  
**Baseline**: Not explicitly reported; results are presented relative to each model family.

## Results  
| Model | Reasoning | Planning | Action | RAG | IF |
|-------|-----------|----------|--------|------|----|
| Anthropic Sonnet 4.5 | 0.849 | 0.435 | 0.734 | 0.860 | 0.828 |
| Anthropic Opus 4.1 | 0.862 | 0.538 | 0.731 | 0.859 | 0.834 |
| Haiku 3.5 | 0.596 | 0.398 | 0.508 | 0.848 | 0.745 |
| OpenAI GPT‑5 | 0.840 | 0.427 | 0.700 | 0.671 | 0.873 |
| Llama 3.3‑70B‑Instruct | Not reported | Not reported | Not reported | Not reported | Not reported |

**Key Findings**  
- Thinking‑enabled models outperform non‑thinking variants by ~8.1 % in Planning and ~12 % in Action.  
- Proprietary models achieve higher, more stable performance than open‑source models; open‑source non‑thinking models sometimes outperform thinking ones in strict output‑control tasks.  
- Top models (Sonnet 4.5, Opus 4.1) reach 85.8 % accuracy on single‑document Reasoning and 84.6 % on multi‑document Reasoning.  
- RAG faithfulness peaks at 0.860 (Sonnet 4.5) and drops to 0.671 (GPT‑5).  
- IF accuracy is highest for GPT‑5 (0.873) and lowest for Haiku 3.5 (0.745).
